# BLZ-template
The bare bones template for FUELPHP websites.

## Dependencies
* PHP
* Composer
  * Fuel (fuel/core)
  * BundleFU (dotsunited/bundlefu)
  * Ice (blazed-labs/ice)

## Running Locally

  To install run,

  ```shell
  npm run install
  ```

  To run a simple development server, simply run:

  ```shell
  npm run dev
  ```
